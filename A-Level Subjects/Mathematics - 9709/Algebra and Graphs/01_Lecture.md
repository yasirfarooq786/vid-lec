## 1. Quadratics

**Standard form:** **ax² + bx + c = 0** (a ≠ 0).

**Discriminant:** **Δ = b² − 4ac**

- **Δ > 0:** two distinct **real** roots  
- **Δ = 0:** one **repeated** real root  
- **Δ < 0:** no **real** roots (complex roots in **P3**)

**Completing the square:** **x² + px = (x + p/2)² − (p/2)²** — reveals **vertex** and helps **sketch**.

**Sum/product of roots:** **α + β = −b/a**, **αβ = c/a** (for **ax² + bx + c = 0**).

**Simultaneous:** **linear + quadratic** → substitute → **discriminant** for **intersection** count (tangent when **Δ = 0** on reduced quadratic).

---

## 2. Inequalities

- **Linear / quadratic:** sketch **graph** or **sign table**; **critical** values where **expression = 0** or **undefined**.  
- **Multiply** by **negative** quantity → **reverse** inequality.  
- **(x − a)(x − b) > 0** → **outside** roots if **a < b** and **positive** **leading** coefficient.

**Modulus:** **|x| < a** ⇔ **−a < x < a**; **|x| > a** ⇔ **x < −a** or **x > a** (a > 0). **|x − a|** = distance from **a** on number line.

---

## 3. Polynomials

**Remainder theorem:** **P(x)** ÷ **(x − a)** has **remainder** **P(a)**.  
**Factor theorem:** **(x − a)** is **factor** ⇔ **P(a) = 0**.

**Algebraic division** or **compare coefficients** for **unknown** constants.

---

## 4. Partial fractions

Express **proper** rational function as **sum** of **simpler** fractions:

- **Distinct linear factors:** **A/(x−1) + B/(x+2)**  
- **Repeated linear:** extra terms **C/(x−1)²**  
- **Irreducible quadratic** (P2/P3): **(Ax+B)/(x²+c)**

**Improper** fraction → **polynomial division** first.

---

## 5. Functions

**Notation:** **f: X → Y**, **domain** **D(f)**, **range** **R(f)** or **Im(f)**.

**Composite:** **fg(x) = f(g(x))** — apply **g** **first** (right to left in notation).

**Inverse f⁻¹:** exists if **one-one** on domain; **graph** is **reflection** of **y = f(x)** in **y = x**; **domain** of **f⁻¹** = **range** of **f**.

**Restricted domain** to force **one-one** (e.g. **f(x) = x²**, **x ≥ 0**).

---

## 6. Graph transformations (**y = f(x)**)

| New graph | Effect |
|-----------|--------|
| **y = f(x) + a** | **Vertical** shift **+a** |
| **y = f(x + a)** | **Horizontal** shift **−a** (“**opposite** sign”) |
| **y = af(x)** | **Vertical** **stretch** **factor a** |
| **y = f(ax)** | **Horizontal** **stretch** **factor 1/a** |
| **y = −f(x)** | **Reflect** in **x-axis** |
| **y = f(−x)** | **Reflect** in **y-axis** |
| **y = \|f(x)\|** | **Reflect** **negative** parts **above** **x-axis** |
| **y = f(\|x\|)** | **Right** half **mirrored** to **left** |

**Asymptotes** move with **transformations** — **recalculate** **intercepts** after shift/stretch.

---

## 7. Exponential and logarithmic graphs (synoptic)

**y = aˣ** (a > 0): **always positive**; **y-intercept 1** if **aˣ** form.  
**y = ln x**: **domain x > 0**; **inverse** of **eˣ**.

**Laws:** **ln(ab) = ln a + ln b**, **ln(a/b) = ln a − ln b**, **ln(aᵏ) = k ln a**.

---

## 8. Differentiation (Pure core — 9709)

**Gradient of curve** **y = f(x):** **dy/dx = f′(x)**

**Standard results:** **d/dx(xⁿ) = nxⁿ⁻¹**; **d/dx(eᵏˣ) = keᵏˣ**; **d/dx(ln x) = 1/x**; **d/dx(sin x) = cos x**, **d/dx(cos x) = −sin x** (x in **radians**).

**Chain rule:** **dy/dx = dy/du × du/dx**  
**Product rule:** **(uv)′ = u′v + uv′**  
**Quotient rule:** **(u/v)′ = (u′v − uv′)/v²**

**Applications:** **tangent** / **normal**; **stationary points** (**f′ = 0**); **second derivative** **d²y/dx²** for **max/min** (**sign** change) or **inflection**.

---

## 9. Integration (Pure core — 9709)

**Antiderivative** + **constant** **+C** (indefinite).

**Definite:** **∫[a,b] f(x) dx = F(b) − F(a)** where **F′ = f**.

**Area** between **curve** and **x-axis** — **sign** of **f** matters; **area** may need **splitting** at **roots**.

**Trapezium rule** (numerical): **½h(y₀ + 2y₁ + … + 2yₙ₋₁ + yₙ)** — **overestimate/underestimate** from **concavity**.

*(Volumes of revolution, integration by parts/substitution — follow **P2/P3** depth.)*

## 10. Examination strategy

- **Show** **factorisation** / **substitution** **clearly** in **“hence”** questions.  
- **Sketch** requires **key** **points**, **asymptotes**, **behaviour** at **±∞**.  
- **Verify** **inverse** by **f(f⁻¹(x)) = x**.  
- In **calculus**, **differentiate** then **substitute** **x** for **gradient**; **integrate** with **correct** **limits** for **area**.

---

## Common mistakes

- **fg ≠ gf** in general.  
- **Horizontal** shift: **f(x+2)** moves graph **left** **2**, **not** right.  
- **log(a + b) ≠ log a + log b**.
